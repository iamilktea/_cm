# 習題 4 (思考題) : 求解 n 次多項式的根

本專案針對 $n \ge 5$ 的高次多項式，實作了一個基於 **數值分析 (Numerical Analysis)** 的通用求解器。

##  思考脈絡與數學原理

### 為什麼不能用公式？
根據 **阿貝爾-魯菲尼定理 (Abel–Ruffini theorem)** 與 **伽羅瓦理論 (Galois Theory)**，五次及以上的多項式**不存在**通用的代數公式解（即無法僅透過加、減、乘、除、開根號來表達根）。

### 解決方案：數值逼近
既然沒有公式，我們改用電腦擅長的「迭代逼近法」。題目提到的「爬山演算法」或「梯度下降」在求解方程式根的特例中，最經典且高效的變體是 **牛頓法 (Newton's Method)**。

我們將求解過程分為兩步：
1.  **尋找一個根 (Newton-Raphson)**：利用函數的導數（切線斜率）快速逼近零點。
2.  **多項式降次 (Polynomial Deflation)**：每找到一個根 $r$，就將原多項式 $P(x)$ 除以 $(x-r)$，得到一個次數更低的新多項式，重複此步驟直到解出所有根。

## 🛠️ 實作邏輯

### 1. 牛頓法 (Newton's Method)
不同於盲目的爬山，牛頓法利用「導數」指引方向，收斂速度極快。
* **迭代公式**：
    $$x_{new} = x_{old} - \frac{f(x_{old})}{f'(x_{old})}$$
* 我們在複數平面上隨機選取初始點，這讓我們能夠找到複數根。

### 2. 綜合除法 (Synthetic Division)
為了避免重複找到同一個根，並簡化問題，我們實作了程式化的綜合除法。
* 當找到根 $r$ 後，計算 $Q(x) = P(x) / (x-r)$。
* 接下來只需對降次後的 $Q(x)$ 求解即可。

### 3. 輸入格式注意
根據題目截圖，輸入陣列 `c` 的格式為 `c[i]` 代表 $x^i$ 的係數。
* 例如 `c = [2, 1, 3]` 代表 $f(x) = 2 + 1x + 3x^2$。
