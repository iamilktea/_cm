## 1. 數學原理

雖然題目給出的是連續傅立葉轉換的積分公式：
$$F(\omega) = \int_{-\infty}^{\infty} f(x)e^{-i\omega x} dx$$

但在電腦科學與數位訊號處理中，我們處理的是有限長度的離散數據，因此必須使用 **離散傅立葉轉換 (DFT)**。積分符號 $\int$ 會變成求和符號 $\sum$。

### DFT (正轉換)
將時域訊號 $x[n]$ 轉為頻域訊號 $X[k]$：
$$X[k] = \sum_{n=0}^{N-1} x[n] \cdot e^{-i \frac{2\pi}{N} kn}$$

### IDFT (逆轉換)
將頻域訊號 $X[k]$ 還原為時域訊號 $x[n]$：
$$x[n] = \frac{1}{N} \sum_{k=0}^{N-1} X[k] \cdot e^{i \frac{2\pi}{N} kn}$$

*注意：逆轉換公式前方有一個 $\frac{1}{N}$ 的正規化因子，這與題目積分公式中的 $\frac{1}{2\pi}$ 扮演相同的角色，確保能量守恆。*

## 2. 程式實作 (`fourier.py`)

### 核心演算法
我們使用最直觀的雙重迴圈實作，時間複雜度為 $O(N^2)$。
1.  **歐拉公式 (Euler's Formula)**：
    程式中使用 `cmath.exp` 來計算複數指數，其數學基礎為：
    $$e^{ix} = \cos(x) + i\sin(x)$$
2.  **`dft(x)`**:
    外層迴圈遍歷頻率 $k$，內層迴圈遍歷時間 $n$，累加所有時間點對該頻率的貢獻。
3.  **`idft(X)`**:
    結構類似，但指數部分的符號為正 ($e^{i...}$)，且最後需除以訊號長度 $N$。

### 驗證邏輯
1.  定義一個原始訊號 $f$。
2.  計算 $F = \text{DFT}(f)$。
3.  計算 $f_{recon} = \text{IDFT}(F)$。
4.  檢查 $f$ 與 $f_{recon}$ 的誤差是否趨近於 0（浮點數誤差範圍內）。
