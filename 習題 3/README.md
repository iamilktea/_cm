
# 習題 3 (加分題) : 解三次多項式的根 (Solving Cubic Roots)

本專案是進階練習，旨在透過 Python 實作**三次方程式**的通解公式，並處理涉及複數運算的複雜情況。

## 📖 題目說明

給定一個三次多項式：

$$f(x) = ax^3 + bx^2 + cx + d$$

我們需要編寫一個函數 `root3(a, b, c, d)` 來求出三個解。

### 數學原理
本程式採用 **通用三次方程式公式 (General Cubic Formula)**，該公式源自卡爾丹諾 (Cardano) 的方法。

為了程式實作方便，我們利用判別參數 $\Delta_0$ 與 $\Delta_1$：

1.  $\Delta_0 = b^2 - 3ac$
2.  $\Delta_1 = 2b^3 - 9abc + 27a^2d$
3.  計算輔助變數 $C$：
    $$C = \sqrt[3]{\frac{\Delta_1 \pm \sqrt{\Delta_1^2 - 4\Delta_0^3}}{2}}$$
4.  最終的三個根由以下公式給出（其中 $\omega$ 為 1 的複數立方根）：
    $$x_k = -\frac{1}{3a} \left( b + \omega^k C + \frac{\Delta_0}{\omega^k C} \right), \quad k \in \{0, 1, 2\}$$

## 🛠️ 實作重點

### 1. 複數運算的處理
三次方程式即使三個根都是實數，中間計算過程仍可能涉及複數（不可約情況）。因此必須全程使用 `cmath` 模組。

* **立方根的處理**：`z**(1/3)` 在 Python 中回傳的是主值（Principal Root），配合上述公式中的 $\omega^k$ 旋轉，即可求出所有解。
