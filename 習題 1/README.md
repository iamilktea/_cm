 assert ... == f(x) 在數學上是正確的，但在電腦科學中，浮點數運算 (Floating-point arithmetic) 會有極小的誤差。
 例如：電腦算出來可能是 3.99999，而原本是 4.0。
 解決方法：將程式碼改為檢查 abs(a - b) < 誤差容忍值 (例如 0.01)。
 integral 函數原理這是使用「矩形法」或「黎曼和」。
 我們把函數圖形切成無數個寬度為 h 的小長方形，將它們的面積加總起來。
 df 函數原理這是導數的定義。$\lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$。
 我們取一個很小的 h (0.0001) 來模擬趨近於 0 的過程。
