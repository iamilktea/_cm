## 1. 主題介紹
本次期中作業選擇的主題是 **曼德博集合 (Mandelbrot Set)**。這是一個在複數平面上生成的碎形 (Fractal) 幾何圖形。它不僅圖形優美，更蘊含了「混沌」與「自我相似」的深奧數學原理，完美結合了我們在課堂上學到的**複數運算**與**迭代**概念。

## 2. 數學原理

曼德博集合的定義非常簡單，基於以下迭代公式：

$$z_{n+1} = z_n^2 + c$$

其中：
* $z$ 是一個複數變數，初始值 $z_0 = 0$。
* $c$ 是複數平面上的一個點（也是複數）。

**收斂與發散的定義：**
對於平面上的每一個點 $c$，我們進行上述迭代：
1.  如果迭代結果 $|z_n|$ 始終保持有限（Bounded），那麼點 $c$ 屬於曼德博集合。
2.  如果迭代結果 $|z_n|$ 趨向無限大（Escapes to infinity），那麼點 $c$ 不屬於該集合。

**著色原理：**
在程式實作中，我們無法迭代無限次。我們設定一個最大迭代次數 `max_iter`。如果在達到這次數前 $|z| > 2$（數學上已證明一旦超過 2 必發散），我們就記錄它「逃逸」時的次數 $n$。
* 屬於集合的點（黑色區域）：直到 `max_iter` 都沒逃逸。
* 不屬於集合的點（彩色區域）：根據逃逸速度 $n$ 給予不同顏色，形成美麗的漸層。

## 3. 程式實作 (fractal.py)

本程式使用 Python 實作，並利用 `numpy` 進行數值計算，`matplotlib` 進行繪圖。

### 核心函數 `mandelbrot(c, max_iter)`
這是程式的靈魂，負責判斷單一複數 $c$ 的命運：
```python
def mandelbrot(c, max_iter):
    z = 0
    for n in range(max_iter):
        if abs(z) > 2: # 判斷是否逃逸
            return n
        z = z*z + c
    return max_iter
